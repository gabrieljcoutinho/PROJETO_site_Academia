# SEJAFIT - Academia & Bem-Estar

O **SEJAFIT** é um projeto de landing page moderno e responsivo para academias, focado em converter visitantes em alunos através de uma interface limpa, dinâmica e motivadora.

## Funcionalidades Principais

* **Design Responsivo:** Interface adaptável para dispositivos móveis, tablets e desktops.
* **Sliders Dinâmicos:** Utilização da biblioteca *Swiper.js* para banners principais, depoimentos e área de blogs.
* **Navegação Inteligente:** Header fixo (sticky) com menu mobile estilo "hamburger" para facilitar o acesso às seções.
* **Seções Estratégicas:**
    * **Home:** Banner de impacto com frases motivacionais.
    * **Sobre:** Detalhes sobre a metodologia e diferenciais da academia.
    * **Serviços:** Exibição visual das modalidades oferecidas (Musculação, Treino para homens/mulheres).
    * **Preços:** Tabela comparativa de planos (Básico e Premium).
    * **Treinadores:** Vitrine dos profissionais especializados com links para redes sociais.
    * **Banner Promocional:** CTA (Call to Action) com destaque para descontos sazonais.
    * **Blog:** Área para postagens de dicas sobre saúde e fitness.

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica do site.
* **CSS3 (main.css):** Estilização personalizada, animações e layout.
* **JavaScript (Vanilla):** Lógica para abertura de menu e interatividade.
* **Font Awesome:** Biblioteca de ícones vetoriais.
* **Swiper.js:** Framework para criação de sliders modernos e leves.

## Instalação e Uso

1.  Clone este repositório ou baixe os arquivos.
2.  Certifique-se de que a estrutura de pastas contenha o diretório `img/` com as fotos necessárias (`home1.jpg`, `sobre.jpeg`, etc.).
3.  Abra o arquivo `index.html` em qualquer navegador moderno.

## Estrutura de Arquivos

* `index.html`: Estrutura principal do site.
* `main.css`: Estilos, fontes e variáveis de cores.
* `main.js`: Inicialização do Swiper e controle do menu responsivo.
* `img/`: Repositório de mídias visuais do projeto.
